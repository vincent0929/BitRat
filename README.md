#打地鼠
###应用简介  

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;打地鼠是一款基于javase开发的小游戏，主要使用的技术有界面绘制、文件操作和多线程技术。其中多线程技术为整个app的核心，界面绘制、线程调度、逻辑处理等都在主线程中实现，老鼠和炸弹的逻辑处理以及爆炸动画等是在各个子线程中实现。
